# Guitar Triad Shapes

**[Visit the site](https://blakeboswell.github.io/triads/)**

Guitar triads — three-note chords — can be voiced three ways (inversions) on any set of three adjacent strings: root position, 1st inversion, or 2nd inversion. That gives 6 distinct moveable shapes per string group (EAD, ADG, DGB, GBe).

Three tabs:

- **Shapes** — all 6 fingering shapes for each string group, for major and minor triads
- **Positions** — all seven diatonic triads of a key, grouped into three hand positions anchored on each inversion of the I chord
- **Fretboard** — fretboard overview

## Implementation

Single self-contained HTML file — no build step, no dependencies.
