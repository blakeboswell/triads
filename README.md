# Guitar Triad Shapes

[Guitar triad shapes](https://blakeboswell.github.io/triads/) — fingering diagrams for every inversion on every string group, organized by key and hand position. Built by Claude.

- **[Shapes](https://blakeboswell.github.io/triads/#shapes)** — all 6 fingering shapes for each string group, for major and minor triads
- **[Positions](https://blakeboswell.github.io/triads/#positions)** — all seven diatonic triads of a key, grouped into three hand positions anchored on each inversion of the I chord
- **[Fretboard](https://blakeboswell.github.io/triads/#fretboard)** — fretboard overview

## Implementation

Single self-contained HTML file — no build step, no dependencies.
