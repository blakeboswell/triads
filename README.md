# Guitar Triad Shapes

An interactive tool for visualizing guitar triads across the fretboard. Built by Claude.

**[Open the tool](https://blakeboswell.github.io/triads/)**

## What it does

Guitar triads — three-note chords — can be voiced three ways (inversions) on any set of three adjacent strings: root position, 1st inversion, or 2nd inversion. That gives 6 distinct moveable shapes per string group (EAD, ADG, DGB, GBe).

The tool has four tabs:

- **Introduction** — explains inversions, positions, and how to read the diagrams
- **Shapes** — all 6 fingering shapes for each string group, for major and minor triads
- **Positions** — all seven diatonic triads (I–vii°) of a selected key, grouped into three hand positions anchored on each inversion of the I chord
- **Fretboard** — fretboard overview

Toggle between **interval labels** (R / 3 / b3 / 5) and **note names** using the button in the top right.

## Implementation

Single self-contained HTML file — no build step, no dependencies.
